Fetch API Practice

How to Use:

This functional webpage has two buttons. The first is to return a list of 5 individuals from an API Fetch request using a FOR LOOP. Select the "Loop Request" button to get started with it.

The second button is a bulk API fetch request. It's interactive as you can specific the number of results you wish to return by inputting a value in the text field below.

Let's get started!

Click either button to perform the FETCH request.
